# AGENTS.md

Syncode is a Node.js/Bun TypeScript CLI that syncs AI agent configurations via adapter-based git + symlink/copy workflows.

## Essentials
- Runtime: Node.js >=20
- Package manager: bun
- Non-standard commands:
  - `bun run dev`
  - `bun run build`
  - `bun run typecheck`
